Проект предоставлен компанией «Цифра».
Постановка задачи
Подготовьте прототип модели машинного обучения для «Цифры». Компания разрабатывает решения для эффективной работы промышленных предприятий.
Модель должна предсказать коэффициент восстановления золота из золотосодержащей руды. В вашем распоряжении данные с параметрами добычи и очистки.
Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.

Содержание
1  Описание данных
2  Подготовка данных
2.1  Проверяю файлы.
2.2  Проверяю, что эффективность обогащения рассчитана правильно.
2.3  Проанализирую признаки, недоступные в тестовой выборке.
2.4  Проведу предобработку данных.
3  Анализ данных
3.1  Посмотрю, как меняется концентрация металлов (Au, Ag, Pb) на различных этапах очистки
3.2  Сравню распределения размеров гранул сырья на обучающей и тестовой выборках
3.3  Исследую суммарную концентрацию всех веществ на разных стадиях
4  Модель
4.1  Напишу функцию для вычисления итоговой sMAPE.
4.2  Обучу разные модели и оцените их качество кросс-валидацией. Выберу лучшую модель и проверьте её на тестовой выборке.
5  Вывод
